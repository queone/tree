# tree

Directory tree printing utility.

```bash
$ tree -?

tree v0.9.0
Directory tree printer utility - https://github.com/queone/tree
Usage:
  tree [options] [directory]

  If no directory is provided, the current directory (".") is used.

Options:
  -f                 Show full file paths. Can be placed after dir path.
  -?, --help, -h     Show this help message and exit

Examples:
  tree
  tree -f /path/to/directory
  tree /path/to/directory -f
  tree -h
```
